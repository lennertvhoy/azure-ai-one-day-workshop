# Course Positioning & Sales Copy

## Headline
Build your own **enterprise-ready RAG chatbot** in one day (Azure edition)

## Subheadline
A hands-on workshop where participants ship a secure, working AI assistant with citations, Key Vault, Managed Identity, and live deployment.

## Core value propositions
- **Ship, don’t just watch:** build and deploy a real RAG chatbot in one day.
- **Enterprise security patterns:** use Key Vault, RBAC, Managed Identity, and safe defaults.
- **Trustworthy answers:** implement retrieval + citations + explicit “I don’t know” behavior.

## Who it’s for
- Technical professionals with basic Azure familiarity
- Developers, architects, IT engineers, technical consultants
- Teams exploring internal copilots / policy assistants / knowledge bots

## Who it’s not for
- Complete beginners with zero cloud/dev exposure
- Learners looking for pure theory (this is build-focused)

## Outcomes by end of day
- Deployed Azure web app with secure secret handling
- Working RAG chatbot answering from docs with citations
- Reusable starter templates (labs + infra + docs)
- Clear path from prototype to production hardening

## Credibility line (optional)
Built with Azure OpenAI + Azure AI Search + FastAPI, using practical enterprise patterns reusable in real environments.
